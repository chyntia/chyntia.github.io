How to Play

The goal of the game is to arrange the tiles sequentially. The fewer moves you used, the higher score you will gain.

Slide Mode

Click any tile which located at the top, bottom, left, or right of the hole to slide it into the hole.

Swing Mode

Click any tile to swing it into the hole. Convenient, isn"t it? But beware, using swing mode will cause you to lose more point than using slide mode. You will also need to slide 10 times to get an additional chance to swing.

Score Calculation

The score is dependent to the maximum number of correct tiles in a state, number of slides you"ve used, and number of swings you"ve used.

score = ( 50 x max_correct ) - ( 1 x num_slide ) - ( 5 x num_swing )
